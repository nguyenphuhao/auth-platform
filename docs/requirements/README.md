# Requirements Package

This folder stores the agreed requirements artifacts for the Auth and User Management Platform.

## Files

- `epic-001-foundation.md`: Foundation prerequisites (US-001 to US-004).
- `epic-002-phone-otp-auth.md`: Phone OTP authentication (US-101 to US-104).
- `epic-003-admin-portal-user-management.md`: Admin portal user management (US-201 to US-205).
- `epic-004-login-history-device-tracking.md`: Login history and device metadata (US-301 to US-304).
- `epic-005-security-safety.md`: Security and safety controls (US-501 to US-502).
- `epic-006-multi-app-reusability.md`: Multi-app deployment and reuse model (US-401).

## Source Alignment

- Product + EPIC requirements from Notion.
- Architecture decisions from `docs/adr/adr-0001-auth-platform-architecture.md`.
- Delivery policy from `AGENTS.md`.
