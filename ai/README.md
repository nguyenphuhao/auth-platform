# AI Workspace

This folder contains project-specific AI Agent assets, separated from product docs and implementation plans.

## Structure
- `ai/skills/`: role skills and orchestrator skills
- `ai/templates/`: reusable prompt templates
- `ai/playbooks/`: workflow guides and role handoff process

## Start Here
1. Read `ai/playbooks/idea-to-delivery-cheatsheet.md`.
2. Use `ai/templates/feature-orchestrator.discovery.prompt.md` for new ideas.
3. Use `ai/playbooks/agent-skill-assignment.md` for role mapping and handoff flow.
