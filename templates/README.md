# Prompt Templates

This folder stores reusable prompt templates for AI Agent workflows.

## Files
- `feature-orchestrator.discovery.prompt.md`
- `feature-orchestrator.discussion.prompt.md`
- `feature-orchestrator.planning.prompt.md`
- `role-handoff.prompts.md`

## Usage
1. Copy a template.
2. Replace placeholders in `<...>`.
3. Send to AI Agent.
